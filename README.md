# Wireshark Network Analysis

## 🎯 Objective
To analyze network traffic using Wireshark and understand different protocols.

## 🛠 Tool Used
- Wireshark

## 🧪 Steps Performed
1. Captured live network traffic
2. Applied filters (DNS, TCP, TLS)
3. Analyzed packets and protocols

## 🔍 Observations
- DNS queries used for domain resolution
- TCP protocol used for communication
- TLS protocol used for secure encrypted traffic
